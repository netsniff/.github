# NetSniff

*Network sniffing tools for deep packet inspection and traffic analysis.*

Every byte on the wire tells a story. We write the tools that read it.
Precision-engineered, zero-fluff packet analysis for engineers who think in hex.
If it crosses a NIC, we capture it, decode it, and make it legible.
